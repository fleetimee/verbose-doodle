Usage Guide

  • currentColor: Logos use currentColor to inherit text color - perfect for matching your design system
  • React Component: Import with import { Logo } from "@/components/ui/logo"
  • Icon Version: Already set as favicon in index.html
  • Wordmark: Use in navigation: <Logo variant="wordmark" size="md" />
  • Theme Support: Pass theme="auto" for automatic dark mode detection
  • Custom Color: Set text color in parent: <div className="text-blue-600"><Logo /></div>
